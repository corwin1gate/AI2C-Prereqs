## ğŸ—ï¸ Mutable vs. Immutable

Python data structures are categorized by whether they can be changed after creation. 

- **Immutable (Tuples):** Once created, the elements and size cannot be changed.  These are useful for fixed data collections.
- **Mutable (Lists):** Can be modified by adding, removing, or changing elements.  These are the most flexible and commonly used structures for dynamic data.

## ğŸ”² Nested Lists

A nested list is a list that contains other lists as its elements. 

- **Arrays:** Nested lists are frequently used to represent multi-dimensional data, such as a 2D grid or a matrix. 
- **Accessing Data:** To retrieve data from a nested list, you use multiple indices (e.g., `matrix[row][column]`).
